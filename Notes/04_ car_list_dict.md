## 二、列表（List）—— 按顺序存一组数据
### 1. 是什么
- 有序、可修改的容器
- 用 `[]` 表示，里面放多个数值/字符串
- 索引从 **0** 开始

### 2. 基本语法
```python
# 创建速度档位列表
speed_gears = [10, 25, 40, 30, 15]
```

### 3. 常用操作
1. **取第几个值（索引）**
   ```python
   speed_gears[0]   # 第一个：10
   speed_gears[2]   # 第三个：40
   speed_gears[-1]  # 最后一个：15
   ```

2. **添加数据**
   ```python
   speed_gears.append(50)  # 末尾加一个应急档位
   ```

3. **切片（取一段）**
   ```python
   speed_gears[1:4]  # 取第2～4个，不包含4
   ```

4. **循环遍历**
   ```python
   for speed in speed_gears:
       print("当前速度：", speed)
   ```

### 4. 小车适用场景
- 多档位速度切换
- 记录历史速度
- 存储路径点距离
- 传感器多次采集的数据

---

## 三、字典（Dict）—— 带名字的键值对数据
### 1. 是什么
- 用 `{}` 表示，结构：`键: 值`
- 键（key）就是“名字”，方便查找
- 可以存不同类型数据：数字、字符串、布尔

### 2. 基本语法
```python
car_config = {
    "speed": 30,
    "direction": "forward",
    "mode": "auto",
    "light": True,
    "max_speed": 50
}
```

### 3. 常用操作
1. **取值**
   ```python
   car_config["speed"]       # 30
   car_config["direction"]   # "forward"
   ```

2. **修改值**
   ```python
   car_config["speed"] = 20
   car_config["mode"] = "manual"
   ```

3. **遍历所有键值**
   ```python
   for key, value in car_config.items():
       print(key, "→", value)
   ```

### 4. 小车适用场景
- 小车完整参数配置
- 一段路径的信息（速度+方向+距离）
- 传感器状态、模式状态

---

## 四、列表 + 字典 组合（实战核心）
### 结构思路
- **列表**：存多段路径
- **字典**：存每一段路径的详细信息

```python
path_plan = [
    {"speed":20, "direction":"forward", "distance":10},
    {"speed":30, "direction":"right",   "distance":5},
    {"speed":25, "direction":"forward", "distance":15}
]
```

### 执行方式
```python
for path in path_plan:
    print("速度：", path["speed"])
    print("方向：", path["direction"])
```

---

## 五、列表 vs 字典 怎么选
| 类型 | 特点 | 适合存什么 |
|------|------|------------|
| 列表 | 有序、靠索引 | 一组同类数据：速度列表、路径点 |
| 字典 | 无序、靠名字 | 一组不同属性：小车配置、单段路径 |

---

## 六、本课易错点
1. 列表索引从 **0** 开始，不是 1
2. 字典取值必须用 **引号**：`car_config["speed"]`，不能写 `speed`
3. 列表用 `[]`，字典用 `{}`，不要混用
4. 遍历字典时要用 `.items()` 才能同时拿到键和值

## 七、小车实战总结
1. **列表** = 小车的“多档位/多段路径”
2. **字典** = 小车的“详细参数单”
3. **列表 + 字典 + 循环** = 完整自动行驶逻辑
